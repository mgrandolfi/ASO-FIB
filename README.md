# ASO-FIB

Coursework repository for **ASO (Administración de Sistemas Operativos)** at the **Facultat d'Informàtica de Barcelona (FIB - UPC)**.

This repository contains practical work, scripts, and supporting material related to systems administration and operating-system-level management.

## Overview

ASO is part of the **Information Technologies** specialization and focuses on the practical administration of operating systems and computing environments.

This repo groups together the practices and work developed during the course, including technical exercises and infrastructure-oriented material.

## Contents

- Practical assignments
- Shell scripts and administration tasks
- Course-related notes
- Supporting documentation
- Additional work related to Hyperscalers

## Tech

- GNU/Linux
- Shell scripting
- System administration tools
- Networking and security basics

## Notes

This is an academic repository for the ASO coursework at FIB.

The structure of the repository follows the practical and deliverable-oriented nature of the subject.
