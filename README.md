# ASO-FIB

Coursework repository for **ASO (Administración de Sistemas Operativos)** at the **Facultat d'Informàtica de Barcelona (FIB - UPC)**.

This repository contains practical work, lab scripts, summaries, and additional course-related material developed during the subject.

## Overview

ASO is focused on the practical administration of operating systems and computing environments.

This repository brings together installation, permissions, and backup-related practices, along with lab preparation material and a comparative study on hyperscalers.

## Repository structure

- `Lab Scripts/` — scripts and supporting material for lab work
- `Pràctica d'instal·lació d'aplicacions - ASO.pdf` — application installation practice
- `Pràctica de Permisos - ASO.pdf` — permissions-related practice
- `Pràctica de backups - ASO.pdf` — backup-related practice
- `Resum ASO - Examen Lab.pdf` — lab exam summary
- `Estudi Comparatiu Hyperscalers - Mateus Grandolfi Albuquerque.pdf` — comparative study
- `README.md` — repository overview
